** Overview

   Read a =.pcap= file and generate a =.dot= file with tcp traffic.

** Install

   Clone the repository:

   #+BEGIN_EXAMPLE
     cd ~
     git clone https://github.com/osiris/tcp2dot
   #+END_EXAMPLE

** Dependences

   - tcpdump
   - graphviz

** Usage

   #+BEGIN_EXAMPLE
     tcp2dot test.pcap | neato -Tpng >test.png
   #+END_EXAMPLE
